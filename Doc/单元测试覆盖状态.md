## 运行方式

```bash
# 运行全部测试
dotnet test XUnitTest/XUnitTest.csproj

# 运行指定类的测试
dotnet test XUnitTest/XUnitTest.csproj --filter "FullyQualifiedName~ApiManagerTests"

# 带详细输出
dotnet test XUnitTest/XUnitTest.csproj -v d
```

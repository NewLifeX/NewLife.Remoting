## M2: 流式调用 (Server-Streaming)

> 代码位置: `NewLife.Remoting/ApiClient.cs`, `ApiServer.cs`, `IEncoder.cs`
> 排序原则: 协议标志 → 元数据检测 → 编码器 → 服务端处理 → SSE → 生命周期 → 客户端 API → 代理

| 编码 | 功能 | 实现 | 测试 | 注释 | 说明 |
|------|------|:----:|:----:|:----:|------|
| M2-1 | **SRMP 协议 Streaming Flag** | ✅ | ✅ | ✅ | Flag 字节 bit5(Streaming) + bit4(EndOfStream) 标记流式帧 |
| M2-2 | **ApiAction.IsStreaming 检测** | ✅ | ✅ | ✅ | 反射检测返回类型 `IAsyncEnumerable<T>` 自动标记流式 |
| M2-3 | **Controller 流式检测** | ✅ | ✅ | ✅ | Process 中通过接口检测而非具体类型，支持值类型和引用类型 |
| M2-4 | **Encoder 流式块编码** | ✅ | ✅ | ✅ | `EncoderBase.EncodeStreamChunk` 预计算长度，SpanWriter 合并写入 |
| M2-5 | **ProcessStream 服务端** | ✅ | ✅ | ✅ | 泛型 Helper `SendStreamItems<T>` 处理值类型/引用类型流，异常时发送错误帧 |
| M2-6 | **HTTP SSE 兼容** | ✅ | ✅ | ✅ | `HttpEncoder.CreateStreamResponse` 输出 `text/event-stream`，浏览器和 curl 可调试 |
| M2-7 | **结束标记帧** | ✅ | ✅ | ✅ | 空 action + 空 data 标记流结束，客户端 yield break |
| M2-8 | **客户端断开自动停止** | ✅ | ✅ | ✅ | CancellationToken 链接 Session.Disposed 轮询，await foreach 通过 `.WithCancellation()` 感知断开，自动停止枚举 |
| M2-9 | **InvokeStreamAsync 客户端** | ✅ | ✅ | ✅ | 返回 `IAsyncEnumerable<T>`，SemaphoreSlim + ConcurrentQueue 收集帧，支持 CancellationToken 中途取消 |
| M2-10 | **ClientBase 代理** | ✅ | ✅ | ✅ | 代理 ApiClient.InvokeStreamAsync |

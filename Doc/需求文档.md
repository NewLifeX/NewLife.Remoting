# RPC 重构需求

> 详见 `Doc/RPC重构需求.md` 完整版本。本文档为开发循环 Agent 入口文件。

## 背景与目标

NewLife.Remoting 已有成熟的 RPC 通信架构（ApiClient/ApiServer），但与主流 RPC 框架（如 gRPC）相比，缺少：
- Server-Streaming 流式调用
- 通用元数据/上下文透传
- 高并发大字符串场景稳定性

## 功能需求

| 编号 | 功能 | 优先级 |
|------|------|:------:|
| F1 | 流式调用 (Server-Streaming) | 🔴 Must |
| F2 | 元数据/Headers 传递 | 🔴 Must |
| F3 | 高并发性能修复 | 🔴 Must |
| F4 | 文档与 README 更新 | 🔴 Must |
